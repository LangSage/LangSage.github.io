LexiLand Russian

Кликабельный план тем для бесплатной мобильной игры по русской «лексике выживания». Это не курс грамматики. Слова запоминаются через звук, картинку, действие и реальные ситуации.

Открыть темы MVP-уровни

Для кого

Для взрослых, которые живут в русскоязычном городе и нуждаются в полезных словах для обычной жизни.

Метод

Аудио + картинка + действие + реальная ситуация. Грамматика спрятана внутри готовых фраз и мини-шаблонов.

Цель

Первые 100–150 полезных слов и фраз, чтобы человек начал понимать язык вокруг себя.

Темы

Нажми на тему, чтобы открыть структуру.

Главная идея

LexiLand Russian должен быть мостом от нуля к первому пониманию. Человек играет пять минут и уходит хотя бы с одной полезной фразой, которую можно применить уже сегодня.
